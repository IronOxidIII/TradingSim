Как запустить проект:

1. Клонировать репозиторий
   git clone https://github.com/IronOxidIII/TradingSim

3. Открыть проект в Android Studio
   Запустить Android Studio
   Нажать Open
   Выбрать папку TradingSim
4. Дождаться Gradle Sync

Android Studio автоматически установит зависимости проекта.

4. Выбрать модуль client
   В верхней панели Android Studio выбрать:
   client

5. Запустить приложение
   Нажать кнопку:
   "Run" и выбрать:
   Android Emulator или подключённое Android-устройство
   Как пользоваться приложением:

Главный экран:
На главном экране доступны разделы:
Портфель
Торговля
История сделок

Экран торговли позволяет:
выбирать актив;
просматривать график цены;
вводить количество;
выбирать плечо;
выполнять покупку и продажу.

Выбор актива:
В верхней части экрана находится горизонтальный список активов:
BTC
ETH
SOL
BNB
XRP
DOGE
Для просмотра остальных активов можно сделать горизонтальный свайп.
После выбора:
обновляется график;
меняется цена актива.

Покупка / продажа:
Для совершения операции:
Ввести количество,
Ввести плечо (1, 2, 5 или 10),
Нажать:
Купить,
Продать,
Приложение валидирует введённые данные.

Портфель:
Экран отображает:
список активов;
количество;
цену;
общую стоимость.

История сделок:
Экран отображает историю операций пользователя.
